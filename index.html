Git para min 15-10-2024

git log: todo o histórico


git diff


para se situar: git status


Para adiocnar mudancas: git add 'nome do projeto'


para commitar: git commit -a -u "FRASE QUE VAI APARECER"


para criar uma nova branch: git checkout -b nome_da_nova_branch


Para commitar na primira vex em uma branch nova: git push --set-upstream origin nome_da_nova_branch


Para a origem com branch existente: git push


Não sei usar o git :(


para cancelar commit: rm -f .git/nome_do_arquivo.lock


Observações

Quando adicionar o arquivo com o "git add .", é obrigatorio usar "git commit -m" para commitar

Quando adicionar estiver em uma branch e quiser salvar, já estando na branch, use: "git push", se for o primeiro commit da branch é necessario usar:"git branch -M main nome_da_nova_branch"

Quando adicionar estiver em uma branch e quiser salvar, já estando na branch, use: "git push"
